# SansVarg AI-Powered Goal System

An AI-driven dashboard for tracking goals, generating daily tasks, and curating learning resources using React and Tailwind CSS.

## Features

- **AI Analysis:** Break down goals into actionable tasks (simulated or API integrated).
- **Resources:** Auto-generated YouTube search keywords.
- **Dashboard:** Track daily task completion.
- **Modern UI:** Built with Tailwind CSS, Glassmorphism, and Lucide Icons.
